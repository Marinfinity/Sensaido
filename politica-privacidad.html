Política de Privacidad

Información sobre cómo se recopilan, utilizan y protegen los datos personales en este sitio web.

1. ¿Qué información se recopila?

En este sitio web se pueden recopilar datos personales del usuario cuando interactúa con determinadas funcionalidades, como formularios de contacto, suscripción a comunicaciones o cualquier otro medio de comunicación habilitado en la página.

Los datos que se pueden recopilar incluyen, entre otros:

  • Nombre
  • Dirección de correo electrónico
  • Dirección IP
  • Información del navegador y sistema operativo

Esta información se utiliza únicamente con el fin de responder a consultas, gestionar comunicaciones o mejorar la experiencia de navegación en el sitio web.

Cuando navegas por este sitio web, también se puede recopilar de forma automática determinada información técnica, como la dirección IP o el tipo de navegador, con fines estadísticos y de mejora del servicio.

2. Consentimiento

¿Cómo se obtiene tu consentimiento?

Cuando proporcionas tus datos personales a través de formularios del sitio web, se entiende que consientes la recopilación y el uso de dicha información para la finalidad específica para la que se solicitan.

En caso de que los datos se soliciten para una finalidad adicional, como el envío de comunicaciones informativas o promocionales, se solicitará previamente el consentimiento expreso del usuario.

¿Cómo puedes retirar tu consentimiento?

Si después de haber otorgado tu consentimiento cambias de opinión, puedes retirarlo en cualquier momento contactando mediante:

  • Correo electrónico: [correo@ejemplo.com]
  • Teléfono: [número de teléfono]

3. Divulgación de la información

Los datos personales del usuario no serán cedidos a terceros, salvo en los siguientes casos:

  • Cuando exista obligación legal.
  • Cuando sea necesario para prestar un servicio solicitado por el usuario.
  • Cuando el usuario haya otorgado su consentimiento expreso.

4. Servicios de terceros

Este sitio web puede utilizar servicios de terceros que recopilan información con fines técnicos o de funcionamiento del sitio, como por ejemplo:

  • Servicios de alojamiento web
  • Herramientas de análisis de tráfico
  • Servicios de correo electrónico

Estos proveedores solo accederán a la información necesaria para prestar sus servicios y están obligados a no utilizarla para otros fines.

En algunos casos, estos proveedores pueden estar ubicados fuera del Espacio Económico Europeo, lo que podría implicar transferencias internacionales de datos que se realizarán conforme a la normativa vigente en materia de protección de datos.

5. Enlaces a sitios externos

Este sitio web puede contener enlaces a páginas web externas. Una vez que el usuario abandona este sitio, el responsable no tiene control sobre las políticas de privacidad de dichos sitios y no se hace responsable de sus prácticas.

Se recomienda revisar las políticas de privacidad de cada sitio web visitado.

6. Seguridad

Se adoptan las medidas técnicas y organizativas razonables para proteger la información personal del usuario y evitar su pérdida, uso indebido, acceso no autorizado, divulgación, alteración o destrucción.

No obstante, ningún sistema de transmisión o almacenamiento de datos en Internet es completamente seguro.

7. Edad mínima

Al utilizar este sitio web, el usuario declara tener al menos 14 años, o contar con la autorización de sus padres o tutores legales para proporcionar datos personales.

8. Cambios en la política de privacidad

El responsable se reserva el derecho de modificar la presente política de privacidad en cualquier momento para adaptarla a novedades legislativas o cambios en el funcionamiento del sitio web.

Se recomienda revisar periódicamente esta página para estar informado de posibles cambios.

9. Derechos del usuario

De acuerdo con el Reglamento General de Protección de Datos (RGPD), el usuario tiene derecho a:

  • Acceder a sus datos personales
  • Solicitar la rectificación de datos inexactos
  • Solicitar la supresión de sus datos
  • Solicitar la limitación del tratamiento
  • Oponerse al tratamiento de sus datos
  • Solicitar la portabilidad de sus datos
